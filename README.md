# Event Management System

## Overview
A console-based Event Management System developed in C++ using Object-Oriented Programming concepts. The application allows administrators and guests to manage and register for events, with data stored in CSV files.

## Features
- User Login and Registration
- Admin Panel
- Guest Panel
- Add Events
- View Events
- Register Participants
- CSV File Storage

## Technologies Used
- C++
- Object-Oriented Programming
- File Handling
- STL (Vector, String, Streams)

## How to Run

```bash
g++ main.cpp -o main.exe
./main.exe
```
